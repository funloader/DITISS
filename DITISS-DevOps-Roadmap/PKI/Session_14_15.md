## 1. Information Technology (IT) Act, 2000

### Overview

* **IT Act, 2000** is the **primary cyber law of India**
* Enacted to provide **legal recognition** to:

  * Electronic records
  * Digital signatures
  * Online transactions
* Amended in **2008** to include cybercrimes & data protection

---

### Objectives of IT Act

* Legal recognition of **e-documents**
* Facilitate **e-commerce & e-governance**
* Define **cyber crimes & penalties**
* Establish **Cyber Appellate Tribunal**

---

### Important Sections (High MCQ Weightage)

| Section | Description                         |
| ------- | ----------------------------------- |
| Sec 43  | Damage to computer systems          |
| Sec 65  | Tampering with computer source code |
| Sec 66  | Computer related offences           |
| Sec 66C | Identity theft                      |
| Sec 66D | Cheating by personation             |
| Sec 67  | Publishing obscene content          |
| Sec 69  | Government surveillance powers      |
| Sec 72  | Breach of confidentiality           |
| Sec 79  | Intermediary liability              |

---

### Digital Signature under IT Act

* Based on **Asymmetric cryptography**
* Uses **hashing**
* Verified using **public key**
* Issued by **Certifying Authority (CA)**

---

### Cyber Crimes Covered

* Hacking
* Identity theft
* Online fraud
* Cyber terrorism
* Obscenity & child pornography

---

### MCQ Points

* IT Act enacted in **2000**
* Amendment year → **2008**
* Indian cyber law = **IT Act**
* Section 66 deals with **computer offences**

## 2. LDAP (Lightweight Directory Access Protocol)

### Overview

* **LDAP is a protocol**, not a directory
* Used to access and manage **directory services**
* Stores **users, groups, devices, policies**

---

### LDAP Characteristics

* Client–server model
* Uses **TCP/IP**
* Hierarchical data structure
* Centralized authentication

---

### LDAP Functions

* Authentication
* Authorization
* User & group management
* Policy enforcement

---

### LDAP Data Structure

* Directory Information Tree (DIT)
* Entries identified using **Distinguished Name (DN)**

---

### MCQ Line

> LDAP is a **protocol**, not a directory service

---

## 3. Active Directory (AD)

### Overview

* **Microsoft implementation** of directory services
* Built on **LDAP**
* Used in **Windows domain environments**

---

### Active Directory Components

* Domain
* Forest
* Tree
* Organizational Unit (OU)
* Domain Controller

---

### Active Directory Services

* User authentication
* Access control
* Group policy enforcement
* Centralized administration

---

### LDAP vs Active Directory (Very Important)

| Feature  | LDAP            | Active Directory    |
| -------- | --------------- | ------------------- |
| Type     | Protocol        | Directory Service   |
| Vendor   | Open standard   | Microsoft           |
| Platform | Cross-platform  | Windows             |
| Usage    | Access protocol | Identity management |

---

### MCQ Traps

* ❌ LDAP is a directory
* ✔ AD uses LDAP
* ✔ AD is Windows-based

## 4. Introduction to Blockchain

### Definition

Blockchain is a **decentralized, distributed ledger** that records transactions in a secure and immutable way.

---

### Key Features

* Decentralization
* Immutability
* Transparency
* Cryptographic security

---

### Blockchain Structure

* Block
* Hash
* Previous block hash
* Timestamp
* Transaction data

---

### Types of Blockchain

| Type       | Description                |
| ---------- | -------------------------- |
| Public     | Open to everyone (Bitcoin) |
| Private    | Permissioned               |
| Consortium | Multiple organizations     |

---

### Blockchain Applications

* Cryptocurrency
* Supply chain
* Healthcare records
* Digital identity
* Smart contracts

---

### Security Mechanisms

* Hashing
* Digital signatures
* Consensus algorithms

---

### MCQ Lines

* Blockchain = **distributed ledger**
* Bitcoin uses **public blockchain**
* Blocks linked using **hash**

---

## 5. Blockchain vs Traditional Database

| Feature      | Blockchain    | Database          |
| ------------ | ------------- | ----------------- |
| Control      | Decentralized | Centralized       |
| Mutability   | Immutable     | Editable          |
| Transparency | High          | Limited           |
| Trust        | Trustless     | Trusted authority |

---

## 6. Exam-Focused Quick Facts

* IT Act governs **cyber law in India**
* LDAP is a **protocol**
* Active Directory is **Microsoft directory service**
* Blockchain ensures **immutability**
* Bitcoin is first blockchain application
